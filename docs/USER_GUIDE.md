## 快速入门

### 注册与登录

1. 访问 FlowMind 首页，自动跳转到登录页面
2. 点击「注册」标签，填写用户名、邮箱、密码
3. 注册成功后自动登录进入工作台

### 工作台概览

工作台展示四个核心统计卡片：
- **工作流**：已创建的工作流总数
- **智能体**：已创建的智能体总数
- **任务**：已创建的任务总数
- **执行次数**：所有工作流的执行次数

下方展示最近执行记录和快速操作入口。

## 工作流编排

### 创建工作流

1. 点击左侧菜单「工作流」
2. 点击右上角「新建工作流」按钮
3. 输入工作流名称和描述
4. 点击「创建」进入工作流编辑器

### 使用工作流编辑器

#### 添加节点

从左侧面板拖拽节点类型到画布区域：

| 节点 | 功能 | 图标 |
|------|------|------|
| LLM 调用 | 调用大语言模型生成内容 | 🤖 |
| 条件分支 | 根据条件表达式进行分支判断 | 🔀 |
| 代码执行 | 运行 Python/JavaScript 代码 | ⚡ |
| 输入节点 | 工作流的输入入口 | 📥 |
| 输出节点 | 工作流的输出出口 | 📤 |
| 工具调用 | 调用外部 API 或工具 | 🔧 |
| HTTP 请求 | 发送 HTTP 请求 | 🌐 |
| 变量操作 | 变量赋值与类型转换 | 📦 |

#### 连接节点

1. 将鼠标悬停在节点右侧的端口上
2. 按住并拖拽到另一个节点左侧的端口
3. 松开鼠标完成连接

#### 配置节点

1. 点击画布上的任意节点
2. 右侧面板显示节点配置选项
3. 根据节点类型配置相应参数：
   - **LLM 节点**：从已配置的模型列表中选择模型、设置 System Prompt、调整 Temperature
   - **HTTP 节点**：设置请求方法、URL、Headers、Body
   - **条件节点**：编写条件表达式，配置比较运算符
   - **代码节点**：选择语言（Python/JavaScript），编写代码逻辑

#### 画布操作

- **缩放**：鼠标滚轮或使用右下角控件
- **平移**：按住画布空白区域拖拽
- **小地图**：右下角显示工作流缩略导航

#### 保存与运行

- 点击工具栏「💾 保存」保存工作流
- 点击「▶ 运行」执行工作流并查看结果

## 模型管理

模型管理让你集中配置所有可用的 LLM 模型，智能体和工作流 LLM 节点从此可以自由选择模型，而非使用硬编码选项。

### 添加模型

1. 点击左侧菜单「🧠 模型管理」
2. 点击右上角「＋ 添加模型」按钮
3. 填写模型配置：

| 字段 | 必填 | 说明 |
|------|------|------|
| 模型名称 | ✅ | 显示名称，如 "GPT-4o" |
| 提供商 | ✅ | 从 9 种预设中选择，或选「自定义」 |
| 模型 ID | ✅ | API 实际标识，如 `gpt-4o`、`claude-3-opus-20240229` |
| API 端点 | | 提供商的 API 地址（自定义时必填） |
| API Key | | 认证密钥（支持显示/隐藏切换） |
| 最大 Tokens | | 默认最大输出 Token 数 |
| 流式输出 | | 是否支持 SSE 流式响应 |
| 多模态/视觉 | | 是否支持图片理解 |
| 描述 | | 模型用途说明 |

4. 点击「创建」完成添加

### 支持的提供商

| 提供商 | 标识 | 典型 API 端点 |
|--------|------|--------------|
| OpenAI | `openai` | `https://api.openai.com/v1` |
| Anthropic | `anthropic` | `https://api.anthropic.com` |
| Google | `google` | `https://generativelanguage.googleapis.com` |
| Azure OpenAI | `azure` | `https://{resource}.openai.azure.com` |
| DeepSeek | `deepseek` | `https://api.deepseek.com` |
| 通义千问 | `qwen` | `https://dashscope.aliyuncs.com` |
| 智谱 GLM | `zhipu` | `https://open.bigmodel.cn` |
| Ollama (本地) | `ollama` | `http://localhost:11434` |
| 自定义 | `custom` | 任意 OpenAI 兼容 API |

### 管理模型

#### 筛选与搜索

- **搜索**：在顶部搜索框输入模型名称关键字
- **筛选**：使用提供商下拉框按类别过滤

#### 启用/停用

- 点击卡片右上角的「启用/停用」按钮切换状态
- 停用后的模型不会出现在智能体和工作流的模型选择列表中
- 已使用该模型的智能体和节点不受影响

#### 编辑模型

- 点击卡片底部「✏️ 编辑」按钮打开编辑弹窗
- 修改任意配置后点击「保存」

#### 删除模型

- 点击卡片底部「删除」按钮
- 确认后永久删除该模型配置

### 在智能体中使用

1. 进入「智能体」页面，点击「新建智能体」
2. 在「模型」下拉框中选择已配置的模型
3. 下拉列表自动加载所有启用的模型，显示格式为 `模型名称 (提供商)`

> 💡 如果尚未配置任何模型，下拉框将显示默认选项（GPT-4、Claude 等），并提示前往模型管理页面添加。

### 在工作流 LLM 节点中使用

1. 在工作流编辑器中拖入一个「LLM 调用」节点
2. 选中节点后，右侧面板的「模型」下拉框自动加载已配置模型
3. 选择模型后配置其他参数（Prompt、Temperature 等）

## 智能体配置

### 创建智能体

1. 点击左侧菜单「智能体」
2. 点击「新建智能体」
3. 填写名称、描述，选择类型和模型
4. 点击「创建」完成

### 智能体类型

| 类型 | 用途 |
|------|------|
| 聊天机器人 | 对话式交互，客服、问答 |
| 助手 | 多轮任务处理，复杂推理 |
| 工具型 | 特定功能的 API 封装 |
| 工作流型 | 驱动工作流执行 |

### 智能体详情

点击智能体卡片进入详情页，可查看：
- 基本信息（类型、模型、状态）
- 模型参数（Temperature、Max Tokens）
- 系统提示词（System Prompt）

### 在线测试

在智能体列表页面点击「💬 对话」，可与智能体实时对话测试。

## 任务调度

### 创建任务

1. 点击左侧菜单「任务调度」
2. 点击「新建任务」
3. 关联工作流或智能体，配置调度参数

### 调度类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 手动 | 需要手动触发运行 | 默认模式 |
| 单次 | 指定时间执行一次 | 定时报告 |
| 周期 | 按固定间隔重复执行 | 每小时检查 |
| Cron | 使用 Cron 表达式 | `0 9 * * *`（每天 9:00） |

### 优先级

| 范围 | 级别 |
|------|------|
| 1-3 | 低优先级 |
| 4-6 | 中优先级 |
| 7-8 | 高优先级 |
| 9-10 | 紧急 |

## 执行监控

### 查看执行记录

1. 点击左侧菜单「执行监控」
2. 展示所有工作流的执行记录
3. 每条记录包含：状态、开始时间、耗时
4. 点击记录查看详细信息

### 执行详情

- **状态**：运行中（⟳）、已完成（✓）、失败（✕）
- **输入数据**：工作流接收到的输入
- **输出数据**：工作流生成的结果
- **节点结果**：每个节点的独立执行结果
- **错误信息**：失败时的错误详情

### 从工作台查看

工作台仪表盘展示最近执行记录的概览，支持快速跳转。
