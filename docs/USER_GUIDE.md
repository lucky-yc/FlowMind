## 快速入门

### 注册与登录

1. 访问 FlowMind 首页，自动跳转到登录页面
2. 点击「注册」标签，填写用户名、邮箱、密码
3. 注册成功后自动登录进入工作台

### 工作台概览

工作台展示四个核心统计卡片：
- **工作流**：已创建的工作流总数
- **智能体**：已创建的智能体总数
- **任务**：已创建的任务总数
- **执行次数**：所有工作流的执行次数

下方展示最近执行记录和快速操作入口。

## 工作流编排

### 创建工作流

1. 点击左侧菜单「工作流」
2. 点击右上角「新建工作流」按钮
3. 输入工作流名称和描述
4. 点击「创建」进入工作流编辑器

### 使用工作流编辑器

#### 添加节点

从左侧面板拖拽节点类型到画布区域：

| 节点 | 功能 | 图标 |
|------|------|------|
| LLM 调用 | 调用大语言模型生成内容 | 🤖 |
| 条件分支 | 根据条件表达式进行分支判断 | 🔀 |
| 代码执行 | 运行 Python/JavaScript 代码 | ⚡ |
| 输入节点 | 工作流的输入入口 | 📥 |
| 输出节点 | 工作流的输出出口 | 📤 |
| 工具调用 | 调用外部 API 或工具 | 🔧 |
| HTTP 请求 | 发送 HTTP 请求 | 🌐 |
| 变量操作 | 变量赋值与类型转换 | 📦 |

#### 连接节点

1. 将鼠标悬停在节点右侧的端口上
2. 按住并拖拽到另一个节点左侧的端口
3. 松开鼠标完成连接

#### 配置节点

1. 点击画布上的任意节点
2. 右侧面板显示节点配置选项
3. 根据节点类型配置相应参数：
   - **LLM 节点**：选择模型、设置 System Prompt、调整 Temperature
   - **HTTP 节点**：设置请求方法、URL、Headers
   - **条件节点**：编写条件表达式
   - **代码节点**：选择语言，编写代码

#### 画布操作

- **缩放**：鼠标滚轮或使用右下角控件
- **平移**：按住画布空白区域拖拽
- **小地图**：右下角显示工作流缩略导航
- **撤销/重做**：工具栏中的 ↩ ↪ 按钮

#### 保存与运行

- 点击工具栏「💾 保存」保存工作流
- 点击「▶ 运行」执行工作流并查看结果

## 智能体配置

### 创建智能体

1. 点击左侧菜单「智能体」
2. 点击「新建智能体」
3. 选择智能体类型和模型
4. 创建后进入详细配置页面

### 智能体类型

| 类型 | 用途 |
|------|------|
| 聊天机器人 | 对话式交互，客服、问答 |
| 助手 | 多轮任务处理，复杂推理 |
| 工具型 | 特定功能的 API 封装 |
| 工作流型 | 驱动工作流执行 |

### 参数配置

- **Temperature**：控制生成随机性，0 = 精确，1 = 创意
- **Max Tokens**：最大输出 Token 数
- **System Prompt**：定义智能体的角色和行为规范

### 在线测试

在智能体列表页面点击「💬 对话」，可与智能体实时对话测试。

## 任务调度

### 调度类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 手动 | 需要手动触发运行 | 默认模式 |
| 单次 | 指定时间执行一次 | 定时报告 |
| 周期 | 按固定间隔重复执行 | 每小时检查 |
| Cron | 使用 Cron 表达式 | 每天 9:00 |

### 优先级

- 1-3：低优先级
- 4-6：中优先级
- 7-8：高优先级
- 9-10：紧急

## 执行监控

### 查看执行记录

1. 点击左侧菜单「执行监控」
2. 展示所有工作流的执行记录
3. 每条记录包含：状态、开始时间、耗时
4. 点击记录查看详细信息

### 执行详情

- **状态**：运行中（⟳）、已完成（✓）、失败（✕）
- **输入数据**：工作流接收到的输入
- **输出数据**：工作流生成的结果
- **节点结果**：每个节点的独立执行结果
- **错误信息**：失败时的错误详情

### 从工作台查看

工作台仪表盘展示最近 10 条执行记录的概览。
